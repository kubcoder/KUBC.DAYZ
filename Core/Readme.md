# KUBC.DAYZ.Core
Базовые сущности используемые в файлах игры.
Для использования инструментов подключите к своему проекту пакет [KUBC.DAYZ.Core](https://www.nuget.org/packages/KUBC.DAYZ.Core/)

## [Вектор](Vector.cs)
Сущность вектора, которая представляет собой три координаты. В сущность встроена возможность расчета дистанции как в пространстве, так и в плоскости.
